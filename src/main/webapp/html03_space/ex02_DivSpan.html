2. Span


=> 줄바꿈이 없는 인라인(inLine) 요소(pre같은 블럭과 반대개념)
=> 다른 텍스트와 구별할 때 사용
=> 배경색은 글자에만 적용됨
=> 외관과 관련된 속성은 Span Tag에 적용됨
=> width,height 의미없음 글자 중심, 개행 없음
